## 1. The "Why" Behind the "How"
Seniors are asked "Why did you use Redux instead of Zustand?" or "Why React Native instead of Flutter?".
- **Wrong Answer:** "Because I like it" or "It's popular."
- **Right Answer:** "Given our team's existing React expertise, the need for deep native integration on iOS, and the requirement for over-the-air updates, React Native was the most cost-effective solution with the fastest time-to-market."

---

## 2. Architectural Justification
Be ready to draw and explain your app's architecture:
- Data flow (State management).
- Network layer (Caching, Retries, Offline support).
- Native bridge vs. JSI (Why you chose one for a specific feature).
- Navigation structure (Nesting, Deep linking).

---

## 3. The Trade-off Matrix
Every technical decision has a cost. Seniority is about knowing that cost.
- **Speed vs. Quality:** When is it okay to accumulate technical debt?
- **Third-party vs. Custom:** When do you build it yourself to avoid dependency hell?
- **Consistency vs. Performance:** When do you break the design system for a critical performance gain?

---

## 4. Handling Critical Failures
Interviewers love "War Stories." Prepare a narrative for:
1. **The Bug:** A critical production crash.
2. **The Discovery:** How you used logs/monitoring to find it.
3. **The Fix:** The immediate patch and the long-term architectural change to prevent it.
4. **The Lesson:** What the team learned.

---

## 5. Soft Skills for Technical Leadership
- **Mentorship:** How do you help junior developers?
- **Code Reviews:** How do you provide constructive feedback without being a bottleneck?
- **Stakeholder Management:** How do you explain technical complexity to a non-technical Product Manager?

---

## 6. Portfolio Checklist
Your portfolio project should demonstrate:
- [ ] **Clean Architecture:** Feature-based structure.
- [ ] **Type Safety:** 100% TypeScript.
- [ ] **Testing:** Unit and Integration tests.
- [ ] **Performance:** No frame drops, optimized images.
- [ ] **Security:** No secrets in code, secure storage.
